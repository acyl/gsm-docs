AT Commands

The modem will respond to the commands detailed below. Parameters applicable to each command are listed with the command description. The defaults shown for each configuration command are those used in the Rockwell factory profile.

A/ - Re-execute Command

A - Answer

Dn - Dial in GSM mode

Dialling a telephone number, together with data transfer, is one of the essential tasks carried out by the modem. This is why simple-to-use commands have been created within the Hayes "AT" set of commands specially for dialling. The character D is always used as a prefix to a dialling command. The required telephone number follows.

Moreover, the following command characters have been defined :
L Last Number The modem redials the last issued number
S=n  Stored A string previously saved in the nonvolatile memory is called up and carried out.
See also the&Zn=xcommand.
Example: Command     ATD353


En - Command Echo

Hn - Disconnect (Hang-Up)

In - Identification

On - Return to On-Line Data Mode

Qn - Quiet Results Codes Control

Sn - Read/Write S-Register

Vn - Result Code Form

Wn - Error Correction Message Control

Xn - Extended Result Codes

&Kn - Flow Control

&V - Display Current Configuration and Stored Profiles (forbidden during GSM connection)

&Wn - Store Current Configuration (forbidden during GSM connection)

&Yn - Designate a Default Reset Profile (forbidden during GSM connection)

&Zn=x - Store telephone number (forbidden during GSM connection)
